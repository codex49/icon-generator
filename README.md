# Icon Generator

[![Build Status](https://travis-ci.org/codex49/icon-generator.svg?branch=master)](https://travis-ci.org/codex49/icon-generator)

_Icon Generator for your apps._

## Getting Started

Install dependencies:

```shell
yarn
```

Start dev server:

```shell
yarn start
```

Build for production:

```shell
yarn build
```
